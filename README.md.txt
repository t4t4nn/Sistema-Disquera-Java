#  appDisquera

Sistema de gestión de disquera desarrollado en **Java**, utilizando **NetBeans**, **MySQL** y conexión a base de datos.

##  Descripción
Este proyecto permite administrar información relacionada con **canciones** y **discos**, implementando operaciones CRUD (Crear, Listar, Buscar, Modificar y Eliminar).

Fue desarrollado como proyecto académico para aplicar conocimientos de programación y bases de datos.

---

##  Funcionalidades

###  Gestión de Canciones
- Ingresar canciones
- Modificar canciones
- Eliminar canciones
- Listar canciones
- Buscar canciones

###  Gestión de Discos
- Ingresar discos
- Modificar discos
- Eliminar discos
- Listar discos
- Buscar discos

---

##  Tecnologías utilizadas
- Java
- NetBeans
- MySQL
- DBeaver
- Maven

---

##  Base de Datos
La aplicación se conecta a una base de datos MySQL llamada:

**Disquera**

Incluye un archivo SQL para facilitar la importación de la base de datos.

---

##  Cómo ejecutar el proyecto

1. Clonar o descargar este repositorio
2. Abrir el proyecto en NetBeans
3. Crear o importar la base de datos en MySQL
4. Verificar la conexión en la clase correspondiente
5. Ejecutar el proyecto

---

##  Configuración de conexión
La conexión está configurada actualmente de esta forma:

`jdbc:mysql://localhost:3306/Disquera`

Usuario por defecto:

`root`

Contraseña por defecto:

`(vacía)`

> Si tu entorno usa otra configuración, deberás modificar la conexión en el código.
